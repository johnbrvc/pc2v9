The scripts in this folder are used to support interactive problems.
pc2_interactive.sh - script to execute interactive submission with interactive validator if not using sandboxes
pc2sandbox_interactive.sh - script to execute interactive submission in a sandbox but interactive validator not in sandbox
pc2validate_interactive.sh - Copy interactive results to pc2 specified location - this is the validator that is used after
	an interactive submission has completed.
